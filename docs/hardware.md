# BitRacer Pro Max 硬體說明

## 系統架構

```
Micro USB ──5V──▶ TP4056 充電 IC ──▶ 14500 鋰電池 (4.2V)
                                        │
                    ┌───────────────────┤
                    ▼                   ▼
            TB6612FNG 馬達驅動    MCP1826 LDO ──3.3V──▶ micro:bit
                    │                   │
                    ▼                   ▼
                  馬達 ◀──PWM── MCU (ATSAMD21G18) ◀──I2C──▶ micro:bit
                                 │        │
              ┌──────────┬───────┼────────┼──────────┐
              ▼          ▼       ▼        ▼          ▼
            蜂鳴器   超音波測距  紅外線   磁式編碼器  1.8" TFT LCD
                    (HC-SR04P) (VCNT2020)            (SPI)
```

## 核心公式

### 紅外線歸一化

```
y = (y_max − y_min) / (AD_max − AD_min) × (AD − AD_min) + y_min
```

### 賽道位置估測（加權平均）

```
Lp = (S1·x1 + S2·x2 + S3·x3 + S4·x4 + S5·x5) / (S1 + S2 + S3 + S4 + S5)
感測器位置權重 x = [-2, -1, 0, 1, 2]
```

### PD 回授控制

```
ΔPWM  = Kp·e[n] + Kd·(e[n] − e[n−1])
PWM_L = PWM_B − ΔPWM
PWM_R = PWM_B + ΔPWM
```

### 超音波測距

```
D = t × 340 / 2   （t 為來回時間，聲速 340 m/s）
```

### 編碼器里程

```
d = (PL + PR) / 2 × (π × 23) / 32   （輪徑 23mm，每圈 32 脈衝）
```
